# Polyglot Student Management Platform

## Overview

A system design project demonstrating a scalable student management platform built using a polyglot microservices architecture.

## Technologies

* Golang
* Java
* Django
* PHP
* AngularJS
* Flutter
* Apache Spark
* Docker
* CI/CD

## Features

* Student Registration
* Authentication
* Course Management
* Attendance Tracking
* Report Generation
* Analytics Dashboard

## Architecture

Flutter App → Golang API Gateway → Django Auth Service / Java Student Service / PHP Report Service → Apache Spark Analytics

## DevOps

* Docker
* Docker Compose
* CI/CD Pipeline
* Monitoring

## Analytics

* Attendance Analysis
* Performance Trends
* Student Statistics
